# Handover Analysis in 5G Networks <sub><sup>5G 네트워크의 핸드오버 분석</sup></sub>
## Purpose <sub><sup>목적</sup></sub>
This project analyses the performance of a 5G network when multiple User Equipment
(UE) Devices simultaneously download data. The study evaluates how network performance
changes with increasing numbers of UEs and with varying traffic loads, determined by packet
size and transmission interval.

이 프로젝트는 여러 사용자 장비(UE)가 동시에 데이터를 다운로드할 때 5G 네트워크의 성능을 분석합니다. 본 연구는 패킷 크기와 전송 간격에 따라 결정되는 UE 수 증가와 트래픽 부하 변화에 따라 네트워크 성능이 어떻게 변화하는지 평가합니다.
